Point of Interest Broker - Help Page

The Point of Interest Broker provides a browser-based interface to the public ZTF alert stream.

Point of Interest allows users to look up alerts and to apply filters to display only a subset of alerts. The alert table can then be downloaded.

Descriptions of Table Values

  • date: The date of the alert
  • candid: ZTF transient candidate, unique to an alert
  • objectID: ZTF identifier for this astronomical object, unique to this object, appears in all alerts for this object
  • jd: observation time at start of exposure, given in Julian date
  • filter: Filter name (g, r, i) derived from fid (1, 2, 3)
  • ra: Right Ascension of candidate; J2000 [deg]
  • dec: Declination of candidate; J2000 [deg]
  • magpsf: Magnitude from PSF-fit photometry measured from difference image [mag]

Available Filters

  • You can filter the columns by entering values and then click the "Filter" button.
  • You can enter a specific value for the columns date, candid, objectId. You can also enter ranges in fields such as "magpf", Enter one of the following:
    1. >18
    2. <18
    3. 18.8 19.4
  • You can also filter the columns by clicking the sort indicators to sort in either ascending or descending order.

Viewing details of a single alert

Clicking on the objectId of an alert in the main table will bring you to that alertâ€™s detail page.
Here you can see a plot of the concatenated light curve, the cutout images, calculated features, and a classification done by cross-matching to existing catalogs.

The three cutouts are taken from the .avro file and display the 1x1 arcmin area surrounding the alert.


Lightcurve data is plotted using a "DC" magnitude calculated from the magpsf value from the ZTF alerts. According to ZTF, "magpsf" reports the flux in the difference image, and is positive by construction.

The alert packets can be combined to construct absolute photometric (DC) lightcurves, for example, for (a)periodic variable sources. Such lightcurves can be computed by combining the nearest reference image catalog magnitude (magnr), differential magnitude (magpsf), and isdiffpos (positive or negative difference image detection) as follows:

magDC = -2.5 log_{10} (10[-0.4magnr]+sign10[-0.4magpsf])
with
sign = 1 if isdiffpos = 't'
sign = -1 if isdiffpos = 'f'

In the light curve plots, upper and lower limit detections are denoted with triangles.

Light Curve Features

Light curve features are derived properties of light curves.
Depending on the number of observations, some features might not be calculated (i.e.: too few observations).
Only features which can be calculated for this light curve are displayed.

Cross-matches

Cross-matches to various astronomical catalogs are displayed.
